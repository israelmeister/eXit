# eXit

A simple text-based adventure game with sprites, made with C++ and SFML.

Inspired by the TV series **Mr. Robot**.

## About

This is my first small project. I am not a graphic designer, so the visuals are basic. The focus was on learning C++ and SFML, not on creating original assets.


## Build

- C++17
- SFML 3.0 required

## How to play
[The source can be seen here](https://www.youtube.com/watch?v=g_Miz2ZqSI4&t=5s)

There are two routes:
**First route:**
- "move the barrel" / "move barrel"
- "enter" / "enter tunnel"
- "read" / "read the note"
- "leave" / "leave tunnel"
- "look" / "look around"
- "get on boat" / "get on the boat"

**Second route:**
- "sit down next to my friend"
- "light" / "light a match"
- "leave" / "stay"
